# Monthly Payment Tracker

A simple React + Vite starter app for tracking monthly mortgage payments and recurring bills.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the dev server:
   ```bash
   npm run build
   npm run dev
   ```

3. Open the app in your browser at the local URL shown by Vite.

## Features

- Add mortgage loan details
- Add recurring bills
- View a monthly payment dashboard
- Mark payments as paid
- Keep payment history in browser storage

## Testing

Run the acceptance-style UI tests with:

```bash
npm test
```

For interactive watch mode:

```bash
npm run test:watch
```
